# Elemental
Elemental is my first game project made in a game engine. Using Godot, this is my first ever 3D game!
The game is a 3D combat game set on an alien world. Fight off goblins and critters, and survive as long as possible.
Currently, the game is in it's earlier development stages. I'm hoping to add a lot more, and am excited to continue
working on this!
